# AircomSrv_Util PowerShell

This repository contains PowerShell scripts used to support the AircomSrv_Util environment.

## Structure

- **Config** – Configuration files (connections, environment settings, etc.)
- **Jobs** – Job-specific scripts or wrappers (scheduled/QU jobs).
- **Logging** – Log output and transcripts.
- **Modules** – Reusable PowerShell modules (`.psm1`) shared by scripts.
- **Scripts** – Main PowerShell scripts intended to be executed directly.
